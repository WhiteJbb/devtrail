# 점검 항목
1. **고아 페이지**: 다른 페이지에서 링크되지 않는 페이지 (index는 제외)
2. **단절된 개념**: 여러 페이지에서 언급되지만 전용 페이지가 없는 주제
3. **누락된 교차 링크**: 관련 페이지 간 [[링크]]가 없는 경우
4. **오래된 내용**: 날짜·버전 기준으로 갱신이 의심되는 페이지
5. **추천 신규 주제**: 현재 wiki가 다루지 않는 중요한 관련 주제

# 출력 형식
각 항목별로:
- **[유형]** 대상 페이지/개념: 문제 내용 → 권장 조치

이슈가 없는 항목은 생략한다. 마크다운 보고서로 출력한다 (코드펜스 없이).
