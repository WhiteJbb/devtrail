# 규칙
- 소스에 실제로 나타난 내용만 사용한다. 지어내지 않는다.
- 원본 복사가 아니라 핵심 개념·방법·트러블슈팅을 정제·통합한다
- 여러 소스의 내용이 겹치면 하나로 합친다
- 구조: # 제목 → 개요(1-2문장) → 핵심 내용(소제목별) → 트러블슈팅/주의사항(있을 때) → 참고 소스
- 한국어 마크다운, frontmatter 없이 # 제목으로 시작
- 분량: 400~1000자 (핵심만, 길게 쓰지 않는다)
- 관련 wiki 페이지가 있으면 본문에서 [[페이지명]] 형식으로 링크한다 (아래 목록에 있는 것만)
